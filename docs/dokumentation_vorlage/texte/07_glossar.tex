\chapter*{Glossar}

\begin{description}
 \item[MySQL]{MySQL ist ein relationales Datenbankmanagmentsystem. Es ist quelloffen und als Opensource verfügbar. Es bildet die Grundlage für dynamische "`Webauftritte"'}. 
 
 Auf dem MySQL-Server, können mehrere Datenbanken erstellt werden. Auf diese kann unter anderem mit PHP zugegriffen werden.
 \item[PHP]PHP ist eine Skriptsprache zur Erstellung dynamischer Webseiten und wird für Webanwendungen verwendet.
 \item[Relationales Datenbanksystem] Es basiert auf einem tabellenorientierten Modell, in dem die Eigenschaften eines real existierenden Objektes im mathematischen Sinne zueinander beschrieben werden. Sie werden als Datensatz bezeichnet.
 
 Durch Primär- und Fremdschlüssel können Beziehungen zwischen Tabellen erstellt werden.
 
\end{description}
